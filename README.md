# Japanese Practice Web

A simple and clean web application designed to help users practice Japanese Hiragana through typing-based exercises.

## Features

* Hiragana to romaji practice
* Clean and minimal user interface
* Real-time answer validation
* Customizable practice settings:

  * Basic Hiragana
  * Dakuten (゛)
  * Handakuten (゜)
  * Combination Kana (きゃ, しゅ, ちょ)
* Responsive design for desktop and mobile
